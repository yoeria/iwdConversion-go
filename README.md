# IWI <-> DDS

## Summary

The \*.iwd files contain directories which then contain *.iwi files

## What it does

The \*.iwi files are to be converted to *.dds

## Notes

Dependency on third party tools (e.g. iwi2dds.exe) is active until better knowledge of conversion
